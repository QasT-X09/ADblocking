# ADblocking

Локальный скрипт для **Android (прямо на телефоне)**, который пытается нажимать кнопки вида `Skip / Пропустить / Закрыть` через Accessibility API в приложении **Auto.js**.

> ⚠️ Важно: используйте только на своём устройстве и с учётом правил конкретного сервиса.

## Что изменилось

Этот репозиторий теперь ориентирован на запуск **без ПК**:
- не нужен `adb`;
- не нужен Python на компьютере;
- запуск прямо в Auto.js на телефоне.

## Файл скрипта

- `scripts/android_skip_helper_autojs.js` — основной on-device скрипт.

## Требования

1. Android-смартфон.
2. Установленный Auto.js (или совместимый форк).
3. Включённый Accessibility Service для Auto.js.

## Как запустить на телефоне

1. Скопируйте `scripts/android_skip_helper_autojs.js` в телефон.
2. Откройте файл в Auto.js.
3. Дайте все необходимые разрешения.
4. Запустите скрипт.

Скрипт:
- каждые ~800мс ищет кликабельные элементы;
- проверяет `text`, `content-desc` и `id` на ключевые слова;
- при совпадении нажимает элемент.

## Ключевые слова

По умолчанию используются:
- `skip`
- `skip ad`
- `skip ads`
- `пропустить`
- `закрыть`
- `close`

При необходимости можно отредактировать `DEFAULT_KEYWORDS` в коде.

## Ограничения

- Скрипт эвристический: интерфейсы приложений часто меняются.
- Не все кнопки доступны Accessibility API.
- Для некоторых приложений/экранов возможны ложные срабатывания.
