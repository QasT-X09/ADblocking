# ADblocking

Небольшой репозиторий с локальным Android-скриптом для автоматизации нажатия кнопок вида **Skip/Пропустить/Закрыть** через `adb`.

> ⚠️ Важно: используйте только на своих устройствах и с учётом правил конкретного сервиса/приложения.

## Что внутри

- `scripts/android_skip_helper.py` — опрашивает UI через `uiautomator dump`, ищет кликабельные элементы по ключевым словам и нажимает по центру кнопки.

## Требования

- Включённый Developer mode + USB debugging на Android.
- Установленный `adb` на ПК.
- Подключённое устройство (`adb devices`).

## Быстрый старт

```bash
python3 scripts/android_skip_helper.py --dry-run --interval 1.0
```

`--dry-run` сначала просто покажет, какие элементы находятся, без нажатий.

Когда всё ок, запусти без `--dry-run`:

```bash
python3 scripts/android_skip_helper.py --interval 1.0
```

Остановка: `Ctrl+C`.

## Полезные флаги

- `--serial <id>` — выбрать конкретное устройство, если подключено несколько.
- `--keyword <слово>` — добавить своё ключевое слово (можно повторять).
- `--max-taps <N>` — остановиться после N нажатий.
- `--dry-run` — только лог, без реальных тапов.

Пример:

```bash
python3 scripts/android_skip_helper.py \
  --serial emulator-5554 \
  --keyword "skip" \
  --keyword "пропустить" \
  --max-taps 10
```
